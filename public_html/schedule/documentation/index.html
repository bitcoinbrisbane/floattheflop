Presentation

Caledonian PRO PHP Calendar is a user friendly, php based and multi-user calendar/scheduling script. It has so many great features like timeline, multiple calendars, shared calendars, event reminder, multiple language support and so on. You can customize it easily and use as a standalone scheduling application or integrate into your own application. You just need a web browser or HTML5 supported mobile device to use it.

Features

  • Multi-language
  • Two-level admnistration
  • Responsive
  • Bootstrap supported
  • Import (iCalendar)
  • Event Icon & Colors
  • Upcoming Events
  • Event Copier
  • RSS Feed
  • Event Share Buttons (AddThis)
  • Move Posts to Another User (User Removing)

Call Upcoming Events

If you do not want to use widgets, you can easily call upcoming event list. It will return array based records.

 

<?php
	$upcList = new caledonian(); 
	$upcList->widgetOn = true;
	$upcList->widgetFetchData = 2;
	$upcList->widgetUser = 0;
	$upcList->widgetDataCount = 5;
	$upcList = $upcList->upcoming(true); # Show 1 hour expiration, default true
?>

 

Returns

array > error = true / false (If there no record found, it will return false)

array > data = Event List (array)

Data Keys

note_id = Event ID

user_id = Author ID

title = Event Title

mynotes = Event Details

add_date = Creation Date

note_date = Event Date

note_icon = Event Icon (Font Awesome)

note_color = Icon Color

ip_address = Creator IP Address

data_type = Data Type -> 1 - Private // 0 - Public

Class Settings

->widgetOn = Will enable user selection and data count

->widgetUser = User base event list, you must kow user IDs or leave 0

->widgetFetchData = Data type selection 0 - Public, 1 - Private, 2 - All Datas

->widgetDataCount = Maximum event list number

How To Write Events to Page

<?php
	if($upcList['error']){
		echo('No Record Found!');
	}else{
		foreach($upcList['error'] as $k=>$v){
			echo('
					Title: '. $v['title'] .'<br>
					Date: '. $v['note_date'] .'<br>
					<hr>
				');
		}
	}
?>

API Usage

You can control of your events from remote server. There 3 API action available right now.

  1. Event List
  2. Add Event
  3. Delete Event

API Calls

  • API url: /api/v1/
  • End Points:
    • lists/
    • add/
    • delete/
  • Data Type: POST
  • Data Return: JSON
  • API Key POST Variable: apiKey

How To Use API Hook?

You can find sample API actions, in api.test.php file. There ready-to-use Curl based class available.

API List

Required POST variables;

$_POST['from'] = List Start Date (YYYY-MM-DD)
$_POST['to'] = List End Date (YYYY-MM-DD)
$_POST['offset'] = Page Offset Value (Numeric)
$_POST['count'] = Page Data Limit (Numeric)

API Add

$_POST['title'] = 'Test From API'; # Event Title
$_POST['event'] = 'Test From API Details';  # Event Detail
$_POST['event_date'] = '2015-07-11';  # Event Date
$_POST['user'] = '083535632f263bc92ff8044e6aad58c7'; # User Public Key
$_POST['ip'] = $_SERVER['REMOTE_ADDR']; # IP Address
$_POST['icon'] = 'fa fa-users'; # Optional
$_POST['color'] = 'magenta'; # Optional

API Delete

$_POST['id'] = 3; # Event ID
$_POST['user'] = '083535632f263bc92ff8044e6aad58c7'; # User Public Key

Example API Call

<?php
		$post_data = ((!is_array($_POST)) ? array():$_POST);
		$post_data['apiKey'] = "YOUR PRIVATE API KEY";
		foreach($post_data as $key=>$value) { $fields_string .= $sep.$key.'='.urlencode($value); $sep='&'; }
		
		$options = array(
			CURLOPT_URL => "https://www.example.com/api/v1/add/",
			CURLOPT_HEADER         => false,    // don't return headers
			CURLOPT_ENCODING       => "",       // handle all encodings
			CURLOPT_USERAGENT      => "Caledonian", // who am i
			CURLOPT_CONNECTTIMEOUT => 120,      // timeout on connect
			CURLOPT_TIMEOUT        => 120,      // timeout on response
			CURLOPT_MAXREDIRS      => 10,       // stop after 10 redirects
			CURLOPT_POST           => true,
			CURLOPT_SSL_VERIFYPEER => false,
			CURLOPT_POSTFIELDS     => $fields_string,
			CURLOPT_FRESH_CONNECT  => 1,
			CURLOPT_HTTPHEADER     => array(
											'Cache-Control: no-cache',
											'Content-length: '.strlen($fields_string)
                                            )
		);
		$ch      = curl_init();
		curl_setopt_array( $ch, $options );
		$content = curl_exec( $ch );
		$err     = curl_errno( $ch );
		$errmsg  = curl_error( $ch );
		$header  = curl_getinfo( $ch );
		curl_close( $ch );

		$header['errno']   = $err;
		$header['errmsg']  = $errmsg;
		$header['content'] = $content;
?>

API Returns

API calls will return RESULT named array key. Here is possible results;

NO_API_KEY_PROVIDED = No API Key

INVALID_API_KEY = Invalid API Key

INVALID_USER_TOKEN = Invalid User Token

INVALID_TITLE = Invalid Event Title

INVALID_DETAILS = Invalid Event Details

INVALID_DATE = Invalid Event Date

EVENT_CREATED = Event Successfully Created

EVENT_NOT_CREATED = Error Occurred While Post a Event

ONLY_POST_REQUESTS = Only POST Data Acceptable

NO_RECORD = There No Record Found

REMOVED = Event Successfully Removed
